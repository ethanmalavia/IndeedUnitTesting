# Security Policy

## Supported Versions

This project is an academic automated test suite. Only the latest version on the `main` branch is maintained.

| Version | Supported |
|---------|-----------|
| latest (main) | Yes |
| older commits | No |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Email the maintainer directly at the address listed on the [GitHub profile](https://github.com/ethanmalavia).
3. Include a description of the vulnerability, steps to reproduce, and potential impact.

You can expect an acknowledgment within **48 hours** and a resolution or status update within **7 days**.

## Scope

This repository contains only test automation code (no production servers, no user data, no credentials). Nevertheless, we take dependency security seriously and aim to keep all dependencies up to date.
